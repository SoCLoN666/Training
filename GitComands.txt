Робота з вітками

1. git clone url - Клонує ремоут репозиторій локально (робиться один раз на самому початку)
2. git status
3. git switch
4. git branch
5. git branch -r   - показати список ремоут віток

Робота з новими змінами у вітці

1. git add - перенести зміни в статус стейдж
2. git commit -m comitname  -  

Синзронізація локальних і ремоут віток

1. git pull - стягую останні зміни з ремоут вітки в локальну
2. git push origin branchname  - відправляє локальні зміни в ремоут репозиторій
